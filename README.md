# GNR-SMU : Granite Ridge SMU Control

Tools and kernel driver for AMD Granite Ridge (Zen 5) SMU management.

## Structure du projet
- `/driver` : Pilote noyau Linux.
- `/tools` : Outils userspace (Moniteur C, scripts Python).
- `/docs` : Documentation technique.

## ⚠ Sécurité
Les outils de ce dépôt manipulent directement le matériel. Utilisez-les avec précaution.
